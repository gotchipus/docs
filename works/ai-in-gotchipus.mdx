### What is an AI Agent?

An AI agent is a system or program designed to autonomously perceive its environment, process information, and take action toward specific goals.
It can range from simple rule-based systems to sophisticated learning entities capable of decision-making and long-term adaptation.

In Gotchipus, each dNFT is powered by its own AI spirit, making it not just a collectible—but a thinking, evolving companion.

**Core Characteristics of AI Agents**
| Trait | Description |
| --- | --- |
| Autonomy | Operates independently without constant human input. Makes decisions based on its programming or learned behaviors. |
| Reactivity | Responds to environmental changes in real time, using sensor data, APIs, or blockchain activity as input. |
| Proactivity | Plans ahead, executes actions toward pre-defined goals, and can predict future states to make strategic decisions. |
| Adaptability | Learns from past experiences or data to improve performance over time using techniques like machine learning or reinforcement learning. |
